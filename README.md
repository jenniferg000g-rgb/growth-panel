# 成长面板 🌱

一站式个人成长工具，帮你养成好习惯、管理任务、规划课表、追踪技能进度。

## 功能

- **📋 今日概览** — 每天打开看看今天的习惯打卡和待办
- **✅ 习惯追踪** — 每日打卡，自动计算连续天数
- **📌 任务清单** — 按高/中/低优先级管理待办
- **📅 每周课表** — 规划固定和可变时间段
- **🎯 技能树** — 设定目标，添加里程碑，追踪进度
- **⏱ 番茄钟** — 内置专注计时器

## 使用

双击 `growth.html` 即可在浏览器打开。所有数据自动保存在浏览器本地。

## 数据管理

在「设置」页面可以：
- **导出** — 下载 JSON 备份文件
- **导入** — 恢复之前的备份
- **重置** — 清空数据

## 技术

纯前端单页应用，HTML + CSS + JavaScript，零依赖。
